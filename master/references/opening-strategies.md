# 开场策略选择

## 核心原理

**公理**：学习动机和认知准备状态决定开场策略的有效性。

**推论**：
1. 有相关背景的学生 → 需要激活已有认知（冲突性开场）
2. 完全陌生的学生 → 需要建立学习动机（动机性开场）
3. 有误概念的学生 → 需要先拆除错误认知（拆除性开场）

---

## 三种开场策略

### 策略 1：冲突性开场

**适用场景**：
- 学生有相关背景
- 锚点验证通过（真懂）
- 学生对主题有一定了解

**目标**：
- 激活已有认知
- 制造认知失调（"我以为我懂，但这个例子解释不了"）
- 引发好奇心

**实施方法**：
1. 抛出反直觉现象
2. 抛出悖论
3. 抛出一个"本阶段结束才能回答"的大问题

**示例**：

```
# 线性代数 - 向量空间
学生背景：学过向量，知道向量加法和数乘

开场：
"你觉得函数可以是向量吗？

比如，f(x) = x^2 和 g(x) = 2x，它们能像向量一样相加吗？
(f + g)(x) = x^2 + 2x，这算是'向量加法'吗？

如果函数是向量，那'向量空间'到底是什么？"

→ 学生会困惑："向量不是箭头吗？函数怎么可能是向量？"
→ 制造认知失调，激发探索欲望
```

```
# 微积分 - 极限
学生背景：学过函数，知道函数的图像

开场：
"你觉得 0.999... 等于 1 吗？

如果等于，为什么看起来有无穷多个 9？
如果不等于，那它们之间的差是多少？

这个问题的答案，就藏在'极限'这个概念里。"

→ 学生会纠结："感觉应该等于，但又说不清为什么"
→ 制造认知失调
```

```
# 概率论 - 条件概率
学生背景：学过基础概率

开场：
"有两个孩子的家庭，已知至少有一个是女孩，那么两个都是女孩的概率是多少？

你可能会说 1/2，但正确答案是 1/3。

为什么？这就是'条件概率'的反直觉之处。"

→ 学生会惊讶："怎么可能是 1/3？"
→ 制造认知失调
```

### 策略 2：动机性开场

**适用场景**：
- 学生完全陌生
- 锚点验证失败（不懂或半懂）
- 学生不知道"为什么要学这个"

**目标**：
- 建立学习动机
- 回答"这个有什么用"
- 让学生看到学习的价值

**实施方法**：
1. 展示实际应用场景
2. 讲述历史故事（这个概念是如何被发明的）
3. 连接到学生的兴趣点

**示例**：

```
# 线性代数 - 矩阵
学生背景：完全没学过线性代数

开场：
"你知道 Google 搜索是怎么工作的吗？

当你搜索'线性代数'，Google 需要从几十亿个网页中找到最相关的。
它用的就是矩阵运算——把每个网页表示成一个向量，然后计算相似度。

矩阵不只是数字的表格，它是描述'关系'的工具。
今天我们就来看看，矩阵是如何描述世界的。"

→ 学生会感兴趣："原来矩阵这么有用"
→ 建立学习动机
```

```
# 微积分 - 导数
学生背景：完全没学过微积分

开场：
"你开车时，看到的速度表显示的是什么？

是'瞬时速度'——某一时刻的速度。

但速度是'距离 / 时间'，如果时间是 0，怎么算速度？
这就是微积分要解决的问题：如何定义'瞬时变化率'。

这个概念叫'导数'，它是整个微积分的基础。"

→ 学生会好奇："对啊，瞬时速度怎么算？"
→ 建立学习动机
```

```
# 概率论 - 贝叶斯定理
学生背景：完全没学过概率论

开场：
"你去医院做体检，检测结果显示你得了某种罕见病。
医生说这个检测的准确率是 99%。

你会有多担心？

实际上，即使检测准确率 99%，你真正得病的概率可能只有 10%。

为什么？这就是'贝叶斯定理'要告诉你的——
如何在不确定的世界中做出理性判断。"

→ 学生会震惊："准确率 99% 还不够？"
→ 建立学习动机
```

### 策略 3：拆除性开场

**适用场景**：
- 学生有误概念
- 锚点验证发现错误理解
- 学生的"已知"会干扰新知识的学习

**目标**：
- 暴露误概念
- 制造认知失调（"我以为是对的，但这个例子说明我错了"）
- 拆除错误认知，为重建做准备

**实施方法**：
1. 提供反例，暴露误概念
2. 让学生自己发现矛盾
3. 引导学生重新思考

**示例**：

```
# 线性代数 - 向量
学生背景：认为"向量就是箭头"

开场：
"你说向量就是箭头，那我问你：

温度场可以用向量表示吗？
比如，房间里每个点的温度，这是向量吗？

如果是，那'箭头'在哪里？
如果不是，那温度场是什么？"

→ 学生会困惑："温度没有方向，怎么是向量？"
→ 暴露误概念："向量 = 箭头"是不完整的理解
→ 引导重建："向量是满足特定运算规则的对象"
```

```
# 微积分 - 导数
学生背景：认为"导数就是斜率"

开场：
"你说导数就是斜率，那我问你：

y = x^2 在区间 [0, 2] 上的导数是多少？

如果导数是斜率，那这个区间的斜率是多少？
是 (4-0)/(2-0) = 2 吗？

但 y = x^2 的导数是 2x，在 x=0 时是 0，在 x=2 时是 4。
哪个才是'斜率'？"

→ 学生会困惑："导数和斜率不一样吗？"
→ 暴露误概念："导数 = 斜率"混淆了局部和全局
→ 引导重建："导数是瞬时变化率，斜率是平均变化率"
```

```
# 概率论 - 独立性
学生背景：认为"独立 = 不相关"

开场：
"你说两个事件独立，就是它们不相关。

那我问你：抛两次硬币，第一次正面，第二次正面，这两个事件独立吗？

你可能会说'独立，因为第一次不影响第二次'。

但如果我告诉你，这是同一枚硬币，而且这枚硬币两面都是正面呢？
现在第一次正面，第二次正面，还独立吗？"

→ 学生会困惑："这……好像不独立了？"
→ 暴露误概念："独立"不只是"不影响"，还要求概率相乘
→ 引导重建："独立的数学定义是 P(A∩B) = P(A)P(B)"
```

## 策略选择流程

```python
def 选择开场策略(学生画像, 主题):
    # 读取学生的先验知识地图
    锚点状态 = 学生画像.prior_knowledge
    
    # 策略 1：检查是否有误概念
    if 锚点状态.misconceptions:
        return "拆除性开场", 锚点状态.misconceptions[0]
    
    # 策略 2：检查是否有可用锚点
    elif 锚点状态.verified_anchors:
        return "冲突性开场", None
    
    # 策略 3：完全陌生
    else:
        return "动机性开场", None
```

## 开场后的衔接

### 冲突性开场后

```
学生回答了开场问题（可能答对，可能答错）

导师：
"这个问题很有意思，对吧？

在这个阶段结束时，你会知道答案。
但在那之前，我们需要先理解几个基础概念……"

→ 制造悬念，保持学习动机
→ 开始传递第一个概念
```

### 动机性开场后

```
学生了解了应用场景

导师：
"所以你看，[主题] 不是抽象的数学游戏，它是解决实际问题的工具。

现在我们来看看，[主题] 的核心思想是什么……"

→ 建立学习动机
→ 开始传递第一个概念
```

### 拆除性开场后

```
学生发现了自己的误概念

导师：
"你发现了吗？'[误概念]' 这个理解在某些情况下是对的，但不完整。

我们需要重新理解 [概念]，从更本质的角度。
让我们从头开始……"

→ 拆除误概念
→ 开始重建正确理解
```

## 常见陷阱

### 陷阱 1：对陌生学生用冲突性开场

**错误做法**：
```
学生：完全没学过微积分
导师："你觉得 0.999... 等于 1 吗？"
学生："呃……我不知道，这是什么？"

→ 学生没有认知基础，无法思考这个问题
→ 制造困惑，而非好奇
```

**正确做法**：
```
学生：完全没学过微积分
导师："你开车时，看到的速度表显示的是什么？是'瞬时速度'……"

→ 从学生熟悉的场景切入
→ 建立学习动机
```

### 陷阱 2：对有误概念的学生直接讲正确答案

**错误做法**：
```
学生：认为"导数就是斜率"
导师："不对，导数是瞬时变化率，不是斜率。"

→ 学生不理解为什么错
→ 误概念没有被拆除，只是被压制
```

**正确做法**：
```
学生：认为"导数就是斜率"
导师："那 y = x^2 在区间 [0, 2] 上的导数是多少？"
学生："呃……2？"
导师："但 y = x^2 的导数是 2x，在 x=0 时是 0，在 x=2 时是 4。哪个才是'斜率'？"
学生："啊……我好像搞混了。"

→ 让学生自己发现矛盾
→ 误概念被拆除
```

### 陷阱 3：开场问题太难

**错误做法**：
```
学生：刚学完向量的定义
导师："你觉得无穷维向量空间存在吗？"

→ 学生完全没有概念
→ 制造挫败感，而非好奇
```

**正确做法**：
```
学生：刚学完向量的定义
导师："你觉得函数可以是向量吗？比如 f(x) = x^2 和 g(x) = 2x……"

→ 学生有一定基础，可以思考
→ 制造认知失调，激发好奇
```

## 与 2.x 版本的区别

| 维度 | 2.x | 3.0 |
|------|-----|-----|
| 开场策略 | 强制冲突性开场 | 根据学生状态选择策略 |
| 误概念处理 | 教学中发现时处理 | 开场时主动暴露并拆除 |
| 学习动机 | 假设学生有动机 | 主动建立学习动机 |
| 策略选择 | 导师凭经验 | 基于锚点验证结果 |
